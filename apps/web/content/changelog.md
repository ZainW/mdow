# Changelog

## v1.0.0

Initial release.

- Markdown rendering with md4x (WASM)
- Syntax highlighting for 30+ languages via Shiki
- Mermaid diagram support
- Light and dark themes
- File tree sidebar
- Drag-and-drop file opening
- Available for macOS, Windows, and Linux
